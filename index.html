An open source cloud deployer


About

Orchestration Station Cloud Deployer(OSCD) automates the deployment process and abstracts away the underlying details about deployment from start to finish. You could deploy multiple images, supporting cross-container communication via automated orchestration tools. Additionally, OSCD provides real-time monitoring of the provided infrastructure for users to view deployment progress and up to date statistics.
